=== LOG INPUT ===
A critical AWS EC2 instance went down. Suggest steps to resolve.

=== GEMINI OUTPUT ===
To resolve a critical EC2 outage:
1. Verify the instance state in AWS Console.
2. Check CloudWatch metrics for CPU, memory, and status checks.
3. Review recent deployments or configuration changes.
4. Attempt to restart the instance.
5. If EBS volume corruption is suspected, detach and attach to a healthy instance for data recovery.
6. Enable Auto-Recovery if not already configured.
